一个新手如何搭建简易博客

title: 一个新手学习如何写博文
date: 2020-02-29 17:50:00
author: Humoro-enough

准备

搭建博客需要的四个工具

  • Hexo:Hexo快速、简洁且高效的博客框架
  • Node.js：建立在Chrome上的Javascript运行引擎
  • Git：一款免费、开源的分布式版本控制系统
  • Github：国内一款面向开发者的云端开发平台，提供代码托管，运行空间，质量控制，项目管理等功能

1安装GIt

GIt官网

找到Download，安装自己对应的系统版本，系统 会自动判定你当前版本，推荐下载，如果没有就自己选择

2.安装Node.js

Node.js 下载地址
解压以后安装在自己想安的目录里，然后配置环境变量，在系统变量中的path变量里将你的Node.js安装地址粘贴上去，保存退出。在cmd命令行中输入source /etc/profile使配置生效，然后输入node -v查看版本信息，出现类似v9.3.0代表安装配置成功

3.安装Hexo

在自己喜欢的位置新建一个文件夹作为博客目录，在该博客目录下打开Git Bash命令行，用npm(已随着Node.js一起安装)安装Hexonpm install -g hexo-cli

4.初始化Hexo文件夹

在博客目录下打开GIt Bash命令行输入以下命令：
hexo init //初始化博客
npm install
hexo -v //此时博客已经创建成功

5.Github账号注册

Github账号注册按照提示完成就行，创建账号后点击头像左侧的+号，——> New repository新建一个仓库，仓库名格式为用户名.GitHub.io

下面将Hexo与Github关联起来

6.修改站点配置文件_config.yml

这是我们要修改的地方

保存站点配置文件

7.安装git部署插件

npm install hexo-deployer-git --save

8.生成SSH密钥（win10可以跳过这步）

cd ~/.ssh //检查本机已存在的ssh密钥
ssh-keygen -t rsa -C"注册GitHub账号的邮箱地址"
然后连续3次回车，最终会生成两个文件在/root/.ssh下，打开目录，找到.ssh/id_sa.pub文件，使用cat id_ras.pub命令来查看密钥内容，win10也可以直接用记事本打开，并将内容复制，打开你的github主页，进入个人设置->SSH and GPG keys->New SSH key,将刚复制的内容粘贴到key那里，title随便填，保存

9.进行git全局配置

git config --global user.name"github用户名"
git config --global user.email"注册github的邮箱"

10.寻找图床

把博文里的图片上传到某一网站，然后获得外部链接，这种网站被成为图床
当博客中有图片时，若是很少的话可以存放在source文件中，但是显然不合理，图片会占据大量的存储的空间
我们就需要找一个地方存放这些图片，然后通过外部链接取得这些图片，图床重要的是稳定速度快，所以我个人选了七牛云

11.更换主题

主题下载

1).从官网下载自己喜欢的主题，然后解压之后放在博客目录的themes里
2).在博客目录的_config.yml文件中可以对网站信息进行修改,将theme: landscape修改成theme: smackdown(自己主题文件名),注意冒号后面有空格
3).smackdown里的_config.yml文件是主题的配置文件，可以对主题进行修改
4).将blog部署到github
hexo clean
hexo g
hexo d

12.发布博文

具体操作我其他文章有详细介绍

现在就可以开始个人博客之旅了

至此博客基本就搭建完毕
